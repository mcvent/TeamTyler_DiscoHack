## 📋 О проекте

Cloud Manager — это полнофункциональный клиент для Yandex Disk, разработанный специально для Linux. Программа предоставляет удобный интерфейс для работы с облачным хранилищем, поддерживает двустороннюю синхронизацию, работу с локальными файлами и многое другое.

### 🎯 Основные возможности

| Возможность                       | Описание                                            |
| --------------------------------- | --------------------------------------------------- |
| ☁️ **Облачное хранилище**         | Полная интеграция с Yandex Disk API                 |
| 📁 **Локальные файлы**            | Управление локальной файловой системой              |
| 🔄 **Двусторонняя синхронизация** | Автоматическая синхронизация с интервалом 30 секунд |
| 🖼️ **Два режима отображения**     | Таблица или иконки — на ваш выбор                   |
| ⌨️ **Горячие клавиши**            | Ctrl+C, Ctrl+V, Delete и другие                     |
| 🗑️ **Корзина Downloads**          | Все скачанные файлы сохраняются в отдельную папку   |
| 🔐 **Безопасное хранение**        | Токен шифруется и хранится локально                 |
| 📋 **Буфер обмена**               | Копирование и вставка файлов между папками          |

## 🚀 Установка

### Требования

- **Python 3.11+**
- **Linux** (Ubuntu/Debian/ALT Linux)
- **pip** и **venv**

### Пошаговая установка

```bash
# 1. Клонирование репозитория
git clone https://github.com/mcvent/TeamTyler_DiscoHack.git
cd TeamTyler_DiscoHack

# 2. Создание виртуального окружения
python -m venv venv
source venv/bin/activate

# 3. Установка зависимостей
cd src
pip install -r requirements-dev.txt

# 4. Запуск
python main.py
```
